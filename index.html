Introduction

Flutter is a development framework for creating cross-platform applications.

Apps developed in Flutter compile down to native code for a given platform, resulting in blazing fast apps, only to be beaten by apps developed natively.⚡

Also my favorite front-end framework. Its an absolute joy to write dart code.

Windows System Requirements

Your system must meet the following minimum requirements to install and run Flutter:

  • Windows 10 or later
  • 1.64GB Disk space (not including disk space for IDE and tools)
  • Windows PowerShell 5.0 or newer (this comes pre-installed with Windows 10 but you should make sure you have full access to it)
  • Git for Windows 2.x

Setup

You need to follow a few steps before you can get started developing your own apps in Flutter.

  • Install Android Studio
  • Install Dart
  • Install Flutter

Update Your Path

If you'd like to run Flutter commands through the terminal, you'll need to update your path. Using your terminal to run Flutter commands is highly recommended since GUI solutions don't provide the same functionality and flexibility.

  • From the Start search bar, enter ‘env’ and select Edit environment variables for your account.
  • Under User variables create a new variable named Path with the full path to flutter\bin as its value

You have to close and reopen any open terminals in order for this change to take effect.

Finally, run:

flutter doctor

This will do some setup and give you instructions on setting up any platform dependencies that may be missing.

Getting Started

First we need to create a Flutter project. Navigate to whichever directory you'd like to store your project files and run:

flutter create demo_app

This will automatically generate all the files you need to immediately run your app and start making changes. You can verify by running:

cd demo_app flutter analyze flutter test flutter run lib/main.dart

The default app generated by Flutter will be a counter app. You can play around with that.

Once you're done, let's finish up our Hello World app. Simply delete all the code in the lib/main.dart file and replace it with the following:

import 'package:flutter/material.dart'; void main() { runApp(const MyApp()); } class MyApp extends StatelessWidget { const MyApp({super.key}); @override Widget build(BuildContext context) { return MaterialApp( title: 'Welcome to Flutter', home: Scaffold( appBar: AppBar( title: const Text('Welcome to Flutter'), ), body: const Center( child: Text('Hello World'), ), ), ); } }

The app should hot-reload by itself and you should see 'Hello World' written in the center of the screen.
